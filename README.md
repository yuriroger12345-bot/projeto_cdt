# 💳 PontoPlay Fortune

Sistema de créditos desenvolvido em Python utilizando Tkinter e SQLite.

O projeto possui:

- Cadastro de usuários
- Painel administrativo
- Sistema de créditos
- Estatísticas do sistema
- Interface gráfica moderna
- Banco de dados local SQLite
